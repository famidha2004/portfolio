Tools & Technologies: Python, TensorFlow / PyTorch, NLP libraries (spaCy, NLTK), LIAMA framework, Streamlit

Description: This innovative project automates the creation of multiple-choice questions (MCQs) from diverse educational content sources, including PDFs, hyperlinks, images, and text files. Leveraging LIAMA for multimodal input processing and conditional GANs for question generation, the system produces relevant MCQs with plausible distractors. Educators can upload content via a simple UI and adjust the complexity level to tailor assessments, significantly reducing manual question-setting effort.

Key Features

  • Accepts PDF, hyperlink, image, and text inputs
  • Generates MCQs with contextually appropriate distractors
  • Adjustable difficulty levels to match curriculum needs
  • User-friendly interface for educators to upload content and generate questions
  • Utilizes LIAMA for enhanced multimodal content analysis

  • Multimodal data extraction and processing
