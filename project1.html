Description: This project simulates an Automated Teller Machine (ATM) system to provide basic banking functionalities. Users can log in securely using an account number and PIN, and perform core operations such as balance inquiry, cash deposit, withdrawal, and PIN change. The application emphasizes backend logic implementation, modular programming, and user input validation. It was built using Python dictionaries and functions to model accounts and transactions efficiently.

Key Features

  • Secure login with account number and PIN
  • Balance inquiry and real-time balance updates
  • Deposit and withdrawal operations with validation
  • PIN change facility
  • Error handling for invalid inputs and insufficient funds
